JavaScript

JavaScript é uma linguagem de programação interpretada que é usada para adicionar interatividade e funcionalidades dinâmicas a páginas da web. Ele é executado no lado do cliente, o que significa que não precisa ser enviado para o servidor para ser processado.

Sendo uma linguagem de programação muito poderosa e versátil, e pode ser usada para criar uma variedade de efeitos e funcionalidades, incluindo:

  • Animações
  • Interações do usuário
  • Jogos
  • Gráficos
  • Aplicativos da web

JavaScript é uma linguagem de programação relativamente fácil de aprender, e existem muitos recursos disponíveis para ajudar você a começar. Se você está interessado em aprender JavaScript, existem muitos recursos disponíveis online, incluindo tutoriais, cursos e livros.

Bibliotecas JavaScript

jQuery

jQuery é uma biblioteca JavaScript amplamente usada para manipulação de DOM, eventos, animações e Ajax. Foi construída em 2006 por John Resig. Ela fornece uma API simples e fácil de usar que torna o desenvolvimento de páginas da web mais rápido e fácil.
