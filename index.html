O QUE É JAVASCRIPT?

JavaScript é uma linguagem de programação interpretada que é usada para adicionar interatividade e funcionalidades dinâmicas a páginas da web. Ele é executado no lado do cliente, o que significa que não precisa ser enviado a um servidor para ser processado.

Sendo uma linguagem de programação muito poderosa e versátil, e pode ser usada para criar uma variedade de efeitos e funcionalidades, incluindo:

  • Animações
  • Interações do usuário
  • Jogos
  • Gráficos
  • Aplicativos da web

Existem muitos recursos disponíveis para ajudar você a começar a aprender JavaScript. Se você está interessado em aprender JavaScript, existem muitos recursos disponíveis online, incluindo tutoriais, cursos e livros, por exemplo a EBAC(Escola Britânica de Artes Criativas), Obra Social Dom Bosco com o curso Programador Web, SENAI e outros.

Bibliotecas JavaScript

jQuery

jQuery é uma biblioteca JavaScript amplamente usada para manipulação de DOM, eventos, animações e Ajax. Foi construída em 2006 por John Resig. Ela fornece uma API simples e fácil de usar que torna o desenvolvimento de páginas da web mais rápido e fácil.

Funcionalidades

  • Manipulação de DOM
  • Eventos
  • Animação
  • Ajax

React

O React é uma biblioteca JavaScript de código aberto para criar interfaces de usuário (UIs). É mantida pelo Facebook e uma comunidade de desenvolvedores.

Funcionalidades

  • IU detalhada/descritiva
  • Virtual DOM
  • Armazenamento Variável(Estado)
  • Renderização do lado do servidor
  • Componentes reutilizáveis

Angular

O Angular é uma estrutura JavaScript de código aberto, mantida pelo Google, para construir aplicações web de página única (SPAs). É uma estrutura abrangente que fornece uma ampla gama de recursos para desenvolver aplicações web complexas e escaláveis.

Funcionalidades

  • Arquitetura de Componentes
  • Data Binding
  • Injeção de Dependência
  • Roteamento
  • Servições

Vue

O Vue.js é uma estrutura JavaScript progressiva para construir interfaces de usuário. É uma estrutura leve, flexível e fácil de aprender, tornando-a uma ótima escolha para iniciantes e desenvolvedores experientes.

Funcionalidades

  • Virtual DOM
  • Data Binding
  • Modelo de Componente Reactivo
  • Diretivas Personalizadas
  • Transições e Animações

Lodash

Lodash é uma biblioteca Javascript que fornece um conjunto de funções para que a pessoa programadora possa trabalhar com mais facilidade em tarefas comuns, como manipular arrays, objetos, strings, números, datas e mais.

Funcionalidades

  • Manipular arrays, objetos, strings, números e datas
  • Validar e criptografar dados
  • Trabalhar com funções e promises
  • E muito mais!

Moment

Moment.js é uma biblioteca JavaScript que facilita o trabalho com datas e horas. Ela fornece uma ampla gama de funções para formatar, manipular e validar datas e horas, tornando o desenvolvimento de aplicações que lidam com tempo muito mais fácil.

Funcionalidades

  • Manipular, formatar e validar datas e horas
  • Conversão entre diferentes fusos horários
  • Trabalhar com durações
  • E muito mais!

Chart

Chart.js é uma biblioteca JavaScript de código aberto para criar gráficos responsivos. Ela é fácil de usar e oferece uma ampla gama de tipos de gráficos, incluindo gráficos de barras, gráficos de linhas, gráficos de pizza e muito mais.

Funcionalidades

  • Criação de gráficos de barra, linhas, pizza, radar, polares, bolhas e espelhamento
  • Manipulação de gráficos

D3

D3.js é uma biblioteca JavaScript de código aberto para manipulação de dados e visualização de dados. Ela permite que você crie gráficos interativos e visualizações de dados complexas usando padrões web como SVG, HTML e CSS.

Funcionalidades

  • Criação de gráficos de barra, linhas, pizza, radar, polares, bolhas e espelhamento
  • Árvores de dendrograma
  • Nuvem de palavras
  • Mapas de calor

Three

Three.js é uma biblioteca JavaScript de código aberto para criar e renderizar gráficos 3D na web. Ela é baseada no padrão WebGL e permite que você crie cenas 3D interativas e realistas usando JavaScript.

Funcionalidades

  • Criação de objetos 3D: Cubos, esferas, cilindros, toros, planos, linhas e pontos
  • Manipulação de objetos 3D
