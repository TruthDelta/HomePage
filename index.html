O QUE É JAVASCRIPT?

JavaScript é uma linguagem de programação interpretada que é usada para adicionar interatividade e funcionalidades dinâmicas a páginas da web. Ele é executado no lado do cliente, o que significa que não precisa ser enviado a um servidor para ser processado.

Sendo uma linguagem de programação muito poderosa e versátil, e pode ser usada para criar uma variedade de efeitos e funcionalidades, incluindo:

  • Animações
  • Interações do usuário
  • Jogos
  • Gráficos
  • Aplicativos da web

Existem muitos recursos disponíveis para ajudar você a começar a aprender JavaScript. Se você está interessado em aprender JavaScript, existem muitos recursos disponíveis online, incluindo tutoriais, cursos e livros, por exemplo a EBAC(Escola Britânica de Artes Criativas), Obra Social Dom Bosco com o curso Programador Web, SENAI e outros.

Bibliotecas JavaScript

jQuery

jQuery é uma biblioteca JavaScript amplamente usada para manipulação de DOM, eventos, animações e Ajax. Foi construída em 2006 por John Resig. Ela fornece uma API simples e fácil de usar que torna o desenvolvimento de páginas da web mais rápido e fácil.

Funcionalidades:

  • Manipulação de DOM
  • Eventos
  • Animação
  • Ajax

React

O React é uma biblioteca JavaScript de código aberto para criar interfaces de usuário (UIs). É mantida pelo Facebook e uma comunidade de desenvolvedores.

Funcionalidades:

  • IU detalhada/descritiva
  • Virtual DOM
  • Armazenamento Variável(Estado)
  • Renderização do lado do servidor
  • Componentes reutilizáveis

Angular

O Angular é uma estrutura JavaScript de código aberto, mantida pelo Google, para construir aplicações web de página única (SPAs). É uma estrutura abrangente que fornece uma ampla gama de recursos para desenvolver aplicações web complexas e escaláveis.

Funcionalidades:

  • Arquitetura de Componentes
  • Data Binding
  • Injeção de Dependência
  • Roteamento
  • Servições

Vue.js

O Vue.js é uma estrutura JavaScript progressiva para construir interfaces de usuário. É uma estrutura leve, flexível e fácil de aprender, tornando-a uma ótima escolha para iniciantes e desenvolvedores experientes.

Funcionalidades:

  • Virtual DOM
  • Data Binding
  • Modelo de Componente Reactivo
  • Diretivas Personalizadas
  • Transições e Animações
